# installX还你一个清爽的安装

相信大家都有被手机自带的软件安装器折磨的情况，各种禁止安装，这种验证和识别，不开启安全模式和开了没区别，针对这种情况有没有什么办法绕过呢？

我们可以使用开源软件`installerX`，这款软件使用拥有这类原生的安装体验，安装速度也不差，并且简洁高效，还可以进行降级安装。

下载地址如下：

https://github.com/iamr0s/InstallerX/releases/download/v1.7/InstallerX_v1.7.24.apk

要想使用这个安装工具，有两套方案，一种是你的手机有root权限，另外一种是通过USB调试或者WiFi调试来进行调用。调试的情况下需要借助工具`shizuku`

下载地址如下：

https://github.com/RikkaApps/Shizuku/releases/download/v13.5.4/shizuku-v13.5.4.r1049.0e53409-release.apk

## Shizuku模式使用

首先需要打开手机的开发者模式，，开启USB调试和WiFi调试，然后打开Shizuku。

> 通过ADB启用Shizuku

针对没有root的设备，我们只能使用USB调试和WiFi调试的方式来启动Shizuku，使用如下命令在Windows的Powershell中运行来启用Shizuku。

启动效果：

<img src="./%E6%89%8B%E6%9C%BA%E8%BD%AF%E4%BB%B6%E5%AE%89%E8%A3%85%E5%BE%88%E9%9A%BE%E7%94%A8%E6%80%8E%E4%B9%88%E5%8A%9E.assets/Screenshot_2024-12-29-16-36-22-861_moe.shizuku.pr.png" alt="Screenshot_2024-12-29-16-36-22-861_moe.shizuku.pr" style="zoom:50%;" />

```shell
adb shell sh /storage/emulated/0/Android/data/moe.shizuku.privileged.api/start.sh
```

如果身边没有电脑怎么办？我们可以通过WiFi进行调试，如果是MIUI或者澎湃OS的设备，请记得把**通知样式修改成原生**。
