# 如何卸载微软商店安装的应用

## 问题

Windows 11中国特供版本有一个非常啥比的功能，在微软商店安装的应用程序无法通过应用商店、应用程序自带的功能进行卸载。

在微软商店中，无法看到卸载操作，如图：

![image-20250312134135460](./%E5%A6%82%E4%BD%95%E5%8D%B8%E8%BD%BD%E5%BE%AE%E8%BD%AF%E5%95%86%E5%BA%97%E5%AE%89%E8%A3%85%E7%9A%84%E5%BA%94%E7%94%A8.assets/image-20250312134135460.png)

同样我们也没有办法在控制面板中找到对应程序，如图：

![image-20250312134330483](./%E5%A6%82%E4%BD%95%E5%8D%B8%E8%BD%BD%E5%BE%AE%E8%BD%AF%E5%95%86%E5%BA%97%E5%AE%89%E8%A3%85%E7%9A%84%E5%BA%94%E7%94%A8.assets/image-20250312134330483.png)

我们在设置的应用中，也无法进行卸载，如图：

![image-20250312134409661](./%E5%A6%82%E4%BD%95%E5%8D%B8%E8%BD%BD%E5%BE%AE%E8%BD%AF%E5%95%86%E5%BA%97%E5%AE%89%E8%A3%85%E7%9A%84%E5%BA%94%E7%94%A8.assets/image-20250312134409661.png)

这就很恶心用户。真不愧是厂商一贯的作风：“翔之大者；喂国喂民”。



## 解决办法

如何解决这个问题？我们只能通过Powershell命令行来进行操作，首先我们必须要知道对应应用程序的完整包名。这是非常关键的一个步骤。

我们首先使用管理员的方式来打开Windows Terminal，也就是终端。

![image-20250312134953870](./%E5%A6%82%E4%BD%95%E5%8D%B8%E8%BD%BD%E5%BE%AE%E8%BD%AF%E5%95%86%E5%BA%97%E5%AE%89%E8%A3%85%E7%9A%84%E5%BA%94%E7%94%A8.assets/image-20250312134953870.png)

使用如下命令查看微软商店安装的所有的uWP应用程序：

```powershell
Get-AppxPackage | Select Name, PackageFullName
```

会返回如图内容：

![image-20250312135103913](./%E5%A6%82%E4%BD%95%E5%8D%B8%E8%BD%BD%E5%BE%AE%E8%BD%AF%E5%95%86%E5%BA%97%E5%AE%89%E8%A3%85%E7%9A%84%E5%BA%94%E7%94%A8.assets/image-20250312135103913.png)

如果想指定的应用包名，例如我这里的ChatGPT，可以使用如下命令：

```powershell
Get-AppxPackage *ChatGPT*
# 或者
Get-AppxPackage | Where-Object { $_.Name -like "*ChatGPT*" }
```

会返回如下内容：

![image-20250312135243701](./%E5%A6%82%E4%BD%95%E5%8D%B8%E8%BD%BD%E5%BE%AE%E8%BD%AF%E5%95%86%E5%BA%97%E5%AE%89%E8%A3%85%E7%9A%84%E5%BA%94%E7%94%A8.assets/image-20250312135243701.png)

只获取应用包名称：

```powershell
Get-AppxPackage *ChatGPT* | Select -ExpandProperty PackageFullName
```

导出所有应用的包名到文本文件：

```powershell
Get-AppxPackage | Select Name, PackageFullName | Out-File "$env:USERPROFILE\Desktop\AppsList.txt"
```

列表里面的信息中存在完整的包名，`PackageFullName`（完整包名），只需要这个完整包名即可，接下来进行卸载操作：

```powershell
Remove-AppxPackage OpenAI.ChatGPT-Desktop_1.2025.41.0_x64__2p2nqsd0c76g0
```

结果如下：

![image-20250312135730267](./%E5%A6%82%E4%BD%95%E5%8D%B8%E8%BD%BD%E5%BE%AE%E8%BD%AF%E5%95%86%E5%BA%97%E5%AE%89%E8%A3%85%E7%9A%84%E5%BA%94%E7%94%A8.assets/image-20250312135730267.png)

这样我们就成功进行卸载了。
